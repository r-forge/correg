Welcome to CorReg project!

This R package Find a structure explaining correlations between covariates and use it to compute linear regression. The correg function computes the linear regression under the constraint of the found structure. It helps to face dimension and correlations issues. CorReg stands for Correlations in Regression.

The project summary page you can find here.

Dependencies : install.packages(c("Rcpp","RcppEigen","Rmixmod","lars","elasticnet","Matrix","mclust","ridge","MASS","data.table"),repos="http://cran.univ-lyon1.fr/")
